## 如何启动?
###一、运行前配置
[后端] 启动四个redis
[后端] 启动rabbitMQ	
[后端] 启动MySQL
[前端]启动MongodDB	
[后端] 启动nacos
e.g. redis 端口默认分别是6379、7000、7001、7002，可自定义。
  MySQL 默认需要"windcraft"数据库，可自定义。注意要改一下用户名和密码。
  nacos 执行nacos官网下载包自带的startup脚本即可，建议是standalone模式，可自定义。
